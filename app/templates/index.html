PDF → Excel (Form Extraction)

Upload a PDF form and download an Excel file with 3 columns: Question Type, English Question/Index Text, English Answer Text.

Output schema:

Question Type
One of: Text Box, Text Area, Date, Number, Radio Button, Checkbox, Checkbox Group, Dropdown, Display, Signature, Group Table, Group
English Question/Index Text
The prompt/label text as it appears in the PDF (translated to English if needed)
English Answer Text
Expected response content (options pipe-separated for option groups)
